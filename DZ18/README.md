
### Домашнее задание "Разворачиваем сетевую лабораторию"

### Цель: Научиться менять базовые сетевые настройки в Linux-based системах;

Использовал технологию Network Namespaces в одной VM Ubuntu (вместо 7 отдельных виртуалок)

Создал 7 пространств имён: inetRouter, centralRouter, office1Router, office2Router, centralServer, office1Server, office2Server

Соединил их виртуальными Ethernet-парами (veth) — получились сетевые линки между роутерами и серверами

Назначил IP-адреса по схеме:

centralServer — 192.168.0.2/28 (шлюз 192.168.0.1)

office1Server — 192.168.2.2/26 (шлюз 192.168.2.1)

office2Server — 192.168.1.2/25 (шлюз 192.168.1.1)

Настроил маршрутизацию:

Включил ip_forward на всех роутерах

Прописал маршруты по умолчанию на каждый сервер

Добавил обратные маршруты на inetRouter до сетей офисов

Обеспечил выход в интернет
